Kaltura API

licensedurl.get
Description:
Get the URL for playing an asset - EPG or media (not available for recording for now).
Input Params
Name Type Description Required Default Value
assetType 1 Asset type V
contentId int Identifier of the content to get the link for(file identifier) V
baseUrl string Base URL for the licensed URLs V
assetId string Asset identifier - relevant only for asset_type = 'epg' 1
startDate bigint The start date of the stream (epoch) - relevant only for asset_type = 'epg' 1
streamType 1 The stream type to get the URL for - relevant only for asset_type = 'epg' 1
Output Type
KalturaLicensedUrl
Example HTTP Hit
http://www.kaltura.com/api_v3/service/licensedurl/action/get
JSON object: 
				

{
	assetType: "value",
	contentId: value,
	baseUrl: "value",
	assetId: "value",
	startDate: value,
	streamType: "value"
}
