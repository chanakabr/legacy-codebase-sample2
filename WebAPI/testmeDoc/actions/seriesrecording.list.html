Kaltura API

seriesrecording.list
Description:
Return a list of series recordings for the household with optional filter by status and KSQL.
Input Params
Name Type Description Required Default Value
filter KalturaSeriesRecordingFilter Filter parameters for filtering out the result - support order by only - START_DATE_ASC, START_DATE_DESC, ID_ASC,ID_DESC,SERIES_ID_ASC, SERIES_ID_DESC 1
Output Type
KalturaSeriesRecordingListResponse
Example HTTP Hit
http://www.kaltura.com/api_v3/service/seriesrecording/action/list
JSON object: 
				

{
	filter: {
		objectType: "KalturaSeriesRecordingFilter",
		orderBy: value
	}
}
