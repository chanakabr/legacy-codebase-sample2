Kaltura API

paymentgatewayprofile.getConfiguration
Description:
Gets the Payment Gateway Configuration for the payment gateway identifier given
Input Params
Name Type Description Required Default Value
alias string The payemnt gateway for which to return the registration URL/s for the household. If omitted – return the regisration URL for the household for the default payment gateway V
intent string Represent the client’s intent for working with the payment gateway. Intent options to be coordinated with the applicable payment gateway adapter. V
extraParameters array of KalturaKeyValue Additional parameters to send to the payment gateway adapter. V
Output Type
KalturaPaymentGatewayConfiguration
Example HTTP Hit
http://www.kaltura.com/api_v3/service/paymentgatewayprofile/action/getConfiguration
JSON object: 
				

{
	alias: "value",
	intent: "value",
	extraParameters: [{
		objectType: "KalturaKeyValue",
		key: "value",
		value: "value"
	}]
}
