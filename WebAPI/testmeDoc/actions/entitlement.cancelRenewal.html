Kaltura API

entitlement.cancelRenewal
Description:
Cancel a household service subscription at the next renewal. The subscription stays valid till the next renewal.
Input Params
Name Type Description Required Default Value
subscriptionId string Subscription Code V
Example HTTP Hit
http://www.kaltura.com/api_v3/service/entitlement/action/cancelRenewal
JSON object: 
				

{
	subscriptionId: "value"
}
