Kaltura API

ottuser.refreshSession
Description:
Returns new Kaltura session (ks) for the user, using the supplied refresh_token (only if it's valid and not expired)
Input Params
Name Type Description Required Default Value
refreshToken string Refresh token V
udid string Device UDID 1
Output Type
KalturaLoginSession
Example HTTP Hit
http://www.kaltura.com/api_v3/service/ottuser/action/refreshSession
JSON object: 
				

{
	refreshToken: "value",
	udid: "value"
}
