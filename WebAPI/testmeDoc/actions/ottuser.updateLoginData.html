Kaltura API

ottuser.updateLoginData
Description:
Given a user name and existing password, change to a new password.
Input Params
Name Type Description Required Default Value
username string user name V
oldPassword string old password V
newPassword string new password V
Output Type
bool
Example HTTP Hit
http://www.kaltura.com/api_v3/service/ottuser/action/updateLoginData
JSON object: 
				

{
	username: "value",
	oldPassword: "value",
	newPassword: "value"
}
