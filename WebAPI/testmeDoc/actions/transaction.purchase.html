Kaltura API

transaction.purchase
Description:
Purchase specific product or subscription for a household. Upon successful charge entitlements to use the requested product or subscription are granted.
Input Params
Name Type Description Required Default Value
purchase KalturaPurchase Purchase properties V
Output Type
KalturaTransaction

http://www.kaltura.com/api_v3/service/transaction/action/purchase
