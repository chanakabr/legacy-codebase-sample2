Kaltura API

announcement.add
Description:
Add a new future scheduled system announcement push notification
Input Params
Name Type Description Required Default Value
announcement KalturaAnnouncement The announcement to be added. timezone parameter should be taken from the 'name of timezone' from: https://msdn.microsoft.com/en-us/library/ms912391(v=winembedded.11).aspx Recipients values: All, LoggedIn, Guests V
Output Type
KalturaAnnouncement
Example HTTP Hit
http://www.kaltura.com/api_v3/service/announcement/action/add
JSON object: 
				

{
	announcement: {
		objectType: "KalturaAnnouncement",
		name: "value",
		message: "value",
		enabled: value,
		startTime: value,
		timezone: "value",
		status: "value",
		recipients: "value",
		id: value
	}
}
