Kaltura API

transaction.updateStatus
Description:
Updates a pending purchase transaction state.
Input Params
Name Type Description Required Default Value
paymentGatewayId string Payment gateway identifier V
externalTransactionId string external transaction identifier V
signature string Security signature to validate the caller is a payment gateway adapter application V
status KalturaTransactionStatus Status properties V
Example HTTP Hit
http://www.kaltura.com/api_v3/service/transaction/action/updateStatus
JSON object: 
				

{
	paymentGatewayId: "value",
	externalTransactionId: "value",
	signature: "value",
	status: {
		objectType: "KalturaTransactionStatus",
		adapterTransactionStatus: "value",
		externalId: "value",
		externalStatus: "value",
		externalMessage: "value",
		failReason: value
	}
}
