Kaltura API

ottuser.loginWithPin
Description:
User sign-in via a time-expired sign-in PIN.
Input Params
Name Type Description Required Default Value
partnerId int Partner Identifier V
pin string pin code V
udid string Device UDID 1
secret string Additional security parameter to validate the login 1
Output Type
KalturaLoginResponse
Example HTTP Hit
http://www.kaltura.com/api_v3/service/appToken/action/add
JSON object: 
				

[
	partnerId: value,
	pin: "value",
	udid: "value",
	secret: "value"
]
