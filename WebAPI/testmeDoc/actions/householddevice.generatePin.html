Kaltura API

householddevice.generatePin
Description:
Generates device pin to use when adding a device to household by pin
Input Params
Name Type Description Required Default Value
udid string Device UDID V
brandId int Device brand identifier V
Output Type
KalturaDevicePin
Example HTTP Hit
http://www.kaltura.com/api_v3/service/householddevice/action/generatePin
JSON object: 
				

{
	udid: "value",
	brandId: value
}
