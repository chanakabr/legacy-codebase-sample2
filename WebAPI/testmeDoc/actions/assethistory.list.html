Kaltura API

assethistory.list
Description:
Get recently watched media for user, ordered by recently watched first.
Input Params
Name Type Description Required Default Value
filter KalturaAssetHistoryFilter Filter parameters for filtering out the result 1
pager KalturaFilterPager Page size and index. Number of assets to return per page. Possible range 5 ≤ size ≥ 50. If omitted - will be set to 25. If a value > 50 provided – will set to 50 1
Output Type
KalturaAssetHistoryListResponse
Example HTTP Hit
http://www.kaltura.com/api_v3/service/assethistory/action/list
JSON object: 
				

{
	filter: {
		objectType: "KalturaAssetHistoryFilter",
		orderBy: value,
		typeIn: "value",
		statusEqual: "value"
	},
	pager: {
		objectType: "KalturaFilterPager",
		pageSize: value,
		pageIndex: value
	}
}
