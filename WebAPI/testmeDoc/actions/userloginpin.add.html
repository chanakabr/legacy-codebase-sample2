Kaltura API

userloginpin.add
Description:
Generate a time and usage expiry login-PIN that can allow a single login per PIN. If an active login-PIN already exists. Calling this API again for same user will add another login-PIN
Input Params
Name Type Description Required Default Value
secret string Additional security parameter for optional enhanced security 1
Output Type
KalturaUserLoginPin
Example HTTP Hit
http://www.kaltura.com/api_v3/service/userloginpin/action/add
JSON object: 
				

{
	secret: "value"
}
