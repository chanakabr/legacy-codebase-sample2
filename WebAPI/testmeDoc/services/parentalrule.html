Kaltura API

Service Name parentalrule
Description
Actions
Name Description
disable Disables a parental rule that was previously defined by the household master. Disable can be at specific user or household level.
disableDefault Disables a parental rule that was defined at account level. Disable can be at specific user or household level.
enable Enable a parental rules for a user
list Return the parental rules that applies for the user or household. Can include rules that have been associated in account, household, or user level. Association level is also specified in the response.
