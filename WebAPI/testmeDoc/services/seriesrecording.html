Kaltura API

Service Name seriesrecording
Description
Actions
Name Description
add Issue a record request for a complete season or series
cancel Cancel a previously requested series recording. Cancel series recording can be called for recording in status Scheduled or Recording Only
delete Delete series recording(s). Delete series recording can be called recordings in any status
deleteByEpgId Delete EPG recording that was recorded as part of series
list Return a list of series recordings for the household with optional filter by status and KSQL.
